About

Inspired by lost time during clinic visits.

When I got to the clinic at most I spend around 2 hours on the queue waiting just to get my file retrieved before I can go see a nurse or doctor. This waiting time is an average for all patients who use public medical institutions.

So I decided to build a Hospital Patient File Management system that allows both patients and hospital staff to spend less time looking for patient files. If the patient is in the system then just one search is all thats needed.

As much this is now built as my portfolio project at Holberton, I am still going to pursue it and see it deployed starting with my local clinic so that the world can see the power of software engineering and I can also do some cool stuff with the time saved from clinic visits.
